1. Класс Book:
    - основные запросы (get, post, put, patch, delete) - page-Controller.

2. Класс Customer:
    - основные запросы (get, post, put, patch, delete) - CustomerController-page.

      get-запрос содержит специальную валидацию для проверки формата id и exception;
      put-запрос содержит exception;
      patch-запрос содержит 2 вида exception.

    - get-запрос с возвратом Dto-класса (3 поля). Содержит специальную валидацию для проверки формата id и exception;
      Прописан в CustomerRestController.

3. Класс Teacher:
    - post-запрос на создание нововго объекта. Содержит встроенную валидацию @NotBlank на поля в Dto-классе, exception
      и ExceptionHandler на MethodArgumentNotValidException.class;

    - get-запрос с возвратом Dto-класса (имени, фамилии, рейтинга учителя) по id. Содержит exception и ExceptionHandler
      на Exception.class(TEACHER_NOT_FOUND);

    - get-запрос учителя с возвратом Dto-класса по названию города. Содержит exception и ExceptionHandler на
      Exception.class(TEACHER_IN_THIS_CITY_NOT_FOUND);

    - get-запрос учителя с возвратом Dto-класса по рейтингу. Содержит SQL-запрос в Repository, специальную валидацию
      на проверку значения рейтинга (1-10) и ExceptionHandler на ConstraintViolationException.class
      ("The number is not within the rating!!!");

    - get-запрос учителя с возвратом Dto-класса по направлению и рейтингу. Содержит SQL-запрос в Repository.
