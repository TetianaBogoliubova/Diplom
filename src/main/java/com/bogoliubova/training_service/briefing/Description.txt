1. Класс Book:
    - основные запросы (get, post, put, patch, delete) - page-Controller.

2. Класс Customer:
    - основные запросы (get, post, put, patch, delete) - CustomerController-page.
      get-запрос содержит специальную валидацию для проверки формата id и exception;
      put-запрос содержит exception;
      patch-запрос содержит 2 вида exception.

    - get-запрос с возвратом Dto-класса (3 поля), содержит специальную валидацию для проверки формата id и exception;
      Прописан в CustomerRestController.

3. Класс Teacher:
    - post-запрос на создание нововго объекта, содержит встроенную валидацию @NotBlank на поля в Dto-классе и exception;
    - get-запрос с возвратом Dto-класса (имени, фамилии, рейтинга учителя) по id, содержит exception;
    - get-запрос учителя с возвратом Dto-класса по названию города, содержит exception;
    - get-запрос учителя с возвратом Dto-класса по рейтингу, содержит SQL-запрос в Repository и специальную валидацию на проверку значения рейтинга (1-10);
    - get-запрос учителя с возвратом Dto-класса по направлению и рейтингуб содержит SQL-запрос в Repository.
